# このソフトウェアについて #

HelloDoubleBuffer20160630は私個人が学習目的で作成したソフトウェアです。

C++/win32でダブルバッファしました。
ちらつきを抑えられました。
しかし、表示が遅いです。

# 開発環境 #

* Windows XP SP3
* C++/Win32
* Visual C++ 2010 Express
